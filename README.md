# Web Kali Linux

A website simulating Kali Linux GUI, base on Vue + TailCSS, keep adding new features~

you can access https://asefg.ir/ to use it
## Quick Start
```
git clone https://github.com/asefg482/kali-linux-simlutor-on-web-vuejs.git
cd My Online KALI
npm install
npm run serve

AND USE
```
